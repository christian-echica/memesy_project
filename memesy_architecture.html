Data Flows

🔐 Auth Flow

1Browser → CloudFront → ALB
2ALB routes POST /api/auth/login → Flask (auth target group)
3Flask validates creds → PostgreSQL
4JWT issued, session stored in Redis

📋 Browse Flow

1Browser → CloudFront → ALB
2ALB → Flask (listing target group)
3Redis cache hit → return instantly
4Cache miss → PostgreSQL → cache → return

📤 Seller Upload Flow

1Flask (listing) saves metadata → PostgreSQL
2Media uploaded → S3
3Celery: thumbnail gen, metadata extract
4Celery invalidates Redis cache

💰 Purchase Flow

1ALB → Flask (payment) → Stripe checkout
2Stripe webhook → Flask publishes → SQS
3Lambda generates signed CloudFront URL per asset
4SES emails buyer access links
5Celery updates seller record, sends sale email
